# WEB-ANCHETAS

Base repository for content and publishing automation for Anchetas Medellin.

## Current structure

- `content/posts/`: blog post drafts and publish-ready articles
- `content/pages/`: website page copy
- `assets/`: images and visual resources

## Goal

Use this repository as the source of truth for website content so new posts can be created, reviewed, and published in a repeatable way.
